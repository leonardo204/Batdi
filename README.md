# 밧디 (batdi)

> **너의 야구 친구.** 야구 얘기, 밧디한테.

**밧디**는 KBO 4개 구단의 페르소나를 가진 AI 챗봇과 대화하는 야구 전문 Agentic Chatbot입니다. **bat + buddy** — 한 손엔 배트, 한 손엔 친구를 들고 시즌을 함께 보냅니다.

실시간 스코어·기록·뉴스·잡담·밈을 한 자리에서 풀고, 대화가 쌓일수록 나만의 친구로 자라납니다.

---

## 누구를 위한 서비스인가

- **응원하는 팀이 있는 KBO 팬** — 내 팀 색깔의 친구가 옆에서 같이 응원해 줬으면 하는 사람
- **숫자보다 감정을 함께 나누고 싶은 사람** — 통계 사이트가 아닌, "오늘 진 거 같이 한탄해 줄 친구"가 필요한 사람
- **사투리·지역 문화의 결까지 살아있는 대화를 원하는 사람** — 부산말로 한탄하고, 충청말로 위로하는 친구

---

## 무엇을 할 수 있나

### 한 번의 대화로 끝나는 게 아니라, 시즌 내내 같이 가는 친구

| 기능 | 설명 |
|------|------|
| **실시간 스코어** | "오늘 우리 팀 어때?" → 진행 중인 경기·이닝·득점 상황을 카드로 |
| **기록 조회** | 선수·팀 스탯, 시즌 누적·세이버 스탯·순위 |
| **뉴스 요약** | KBO 관련 최신 뉴스를 팀 페르소나 톤으로 압축 |
| **잡담 / 응원 / 한탄** | 이겼을 땐 같이 폭발, 졌을 땐 같이 한숨 |
| **밈 응답** | 팀별 문화 코드·이모지·이미지로 분위기 살리기 |
| **레벨 · XP** | 대화가 쌓이면 레벨이 오르고, 친밀도가 올라갈수록 자연스러운 톤 |
| **커스텀 페르소나** | 기본 캐릭터 위에 "나만의 말투" 500자 덧붙이기 |
| **관심 선수 알림 (P4+)** | 응원 선수의 출장/기록을 푸시로 |

## 4개 팀, 4개의 친구

각 구단마다 **연고지 사투리 · 지역 음식 문화 · 팀 밈 · 감정 패턴**이 모두 다른 별개의 캐릭터로 설계되었습니다. 같은 질문이어도 어느 팀을 골랐냐에 따라 완전히 다른 친구가 답합니다.

### 한화 이글스 — "새 시대의 독수리"
2025년 준우승의 자신감, 충청도 사투리(`~유`, `그려`), 강력한 타선에 대한 자부심.
> "역시! 올해는 진짜 되는 거여~!! 타선 미쳤어유!!"

### 두산 베어스 — "잠실의 여유"
가을야구를 수도 없이 본 베테랑 팬의 느긋한 서울말. 져도 크게 흔들리지 않음.
> "뭐 이 정도면 괜찮지~ 가을 되면 알아."

### 기아 타이거즈 — "광주 열혈 응원단장"
전라도 사투리(`겁나`, `당께`, `허맛나!`)와 큰 감정 기복. 격정적인 응원과 분노의 한탄.
> "쥑이네! 겁나 잘했당께!"

### 롯데 자이언츠 — "부산의 자존심"
한과 정이 공존하는 부산 사나이. 지면 한탄하면서도 끝까지 응원.
> "마 롯데가 지면 쏘주가 땡기네 카이~"

> 모든 페르소나는 **일베/혐오/자학 표현을 가드레일에서 차단**합니다. 응원과 기대를 기본 톤으로.

상세 페르소나 설계 → [docs/plan/batdi-service-plan.md §4](docs/plan/batdi-service-plan.md)

## 설계 철학

### 1. 팩트는 LLM이 만들지 않는다
숫자(타율·득점·순위)는 **DB에서 직접 바인딩**합니다. LLM은 감정과 문장만 만들고, 사실 검증은 시스템이 보장합니다 — 환각이 끼어들 자리가 없습니다.

### 2. 친구는 한 명이지만, 친구가 본 너는 한 사람이다
사용자마다 **전용 Personal Agent**가 붙어 대화 이력·관심 선수·말투 학습을 전담합니다. 어제 한 얘기를 오늘도 기억하는 친구.

### 3. 빠르고 가벼운 대화
4단계 캐시(L0~L3) 구조로 자주 묻는 질문은 LLM을 거치지 않습니다. "오늘 스코어?" 같은 질의는 0회 LLM 호출로 즉답.

### 4. 가볍게 시작해서 깊게 간다
**MVP 베타 100명**을 시작점으로, 사용자 반응을 보며 기능과 페르소나를 다듬어 갑니다.

| **P0** 학습·PoC | 1~2주 | CopilotKit ↔ LangGraph ↔ A2UI ↔ Gemini E2E 검증 |
